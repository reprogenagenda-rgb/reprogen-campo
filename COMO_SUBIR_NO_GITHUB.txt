COMO SUBIR A CORREÇÃO V07

1. No GitHub, entre no repositório reprogen-campo.
2. Clique em Add file > Upload files.
3. Arraste todos os arquivos extraídos deste pacote.
4. Confirme que a pasta icons/ foi junto.
5. Faça commit com:
   Corrige script visivel no app e atualiza PWA V07
6. Aguarde 1 a 3 minutos.
7. Abra o link github.io.
8. Se o celular mostrar versão antiga:
   - remova o app antigo da tela inicial;
   - limpe cache/dados do site;
   - abra novamente pelo github.io.

Critério de aprovação:
- não aparecer código JavaScript no rodapé;
- abrir abas D0, IATF, DG, M. Natural e Base;
- ícone aparecer ao adicionar na tela inicial;
- funcionar offline após primeiro acesso online.
